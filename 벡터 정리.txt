게임에서의 벡터

특정 개체의 이둉방향과 속도를 표현가능
벡터의 덧셈,뺄셈으로 충돌이나 방향전환이 가능
벡터의 스칼라곱으로 특정방향으로의 
움직임에 대한 가속도

스칼라 == 크기만 있고 방향을 가지지 않는 양



속력은 크기만
속도는 크기랑 방향

(a-b)(a-b) aa-2ab+bb
(a+b)(a+b) aa+2ab+bb

두 벡터가 서로 같을조건

크기 와 방향이 같다 (위치에 상관없다)



벡터의 덧셈에 관한 성질
1) 교환법칙 a + b = b + a
2) 결합법칙 ( a + b ) + c  = a + ( b + c )
3) a + 0 = 0 + a = a
4) a + (-a) = (-a) + a

a-b (oa)-(ob) = ba

코사인 법칙
  
aa = bb + cc - 2bc * cos(A)
bb = cc + aa - 2ca * cos(B)
cc = aa + bb - 2ab * cos(C)

cos(A) = (bb+cc-aa) / 2bc
